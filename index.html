// Questions
How does Evergreen find updates?

It checks what's installed and matches it against a curated database of the latest versions — no guesswork, no scraping random sites.

Does it install updates automatically?

Only if you want it to. The default is Download-only mode: Evergreen fetches the update and you click Update when you're ready.

Is it a subscription?

No. One-time purchase, yours forever. Free updates within the same major version.

How many Macs can I use it on?

Up to 2 Macs you own, with a single license.

What about my privacy?

Everything runs on your Mac. No account, no telemetry about which apps you have — it only reaches out to fetch version info and downloads.

Can I get a refund?

Yes — a 14-day money-back guarantee, no questions asked, handled by Paddle.
