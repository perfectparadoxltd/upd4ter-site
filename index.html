// Questions
How does Evergreen find updates?

It scans the apps installed on your Mac and matches each one against a curated database of the latest released versions — apps from the web, Homebrew casks and developer update feeds, not just the Mac App Store. No guesswork, no scraping sketchy sites.

Does it install updates automatically?

You're in control. The default is Download-only mode: Evergreen fetches the new version and you click Update when it suits you. Prefer hands-off? Turn on automatic updates per app or for everything.

What kinds of apps does it cover?

The apps the Mac App Store can't update for you — direct downloads, indie tools, developer apps, Homebrew casks. If you installed it from a website, Evergreen most likely tracks it.

Is it a subscription?

No. One-time purchase, yours forever. You get free updates within the same major version (all of 1.x). Major upgrades may be paid, and only if you choose to take them.

Is it safe? Does it check what it downloads?

Yes. Before installing, Evergreen verifies the developer's code signature, so you never run a tampered or fake update. It only downloads from each app's official source.

How many Macs can I use it on?

Up to 2 Macs you own, with a single license. Enter the key on each and you're set.

What about my privacy?

Everything runs on your Mac. No account, no sign-up, and zero telemetry about which apps you have installed. The only network activity is fetching version info and the updates themselves.

Which macOS do I need?

macOS 14 Sonoma or later, on both Apple Silicon and Intel Macs.

Can I get a refund?

Absolutely — a 14-day money-back guarantee, no questions asked. Payments are handled by Paddle; just reply to your receipt and we'll sort it out.
